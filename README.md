# 🧠 VoraSense: Adaptive Voice-First Learning

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Open%20Website-brightgreen?style=for-the-badge&logo=vercel)](https://Pragathija.github.io/vorasenseai/)

**VoraSense** is an AI-powered educational platform designed for inclusive learning. It leverages advanced voice recognition and adaptive ML algorithms to create personalized experiences for visually impaired and differently-abled students.

---

## 🌟 Key Features

- **🎙️ Voice-First Interaction**: Full hands-free navigation and learning through natural voice commands.
- **🧠 Adaptive AI Engine**: Real-time confusion detection and dynamic difficulty adjustment.
- **🌍 Multilingual Support**: Content and voice tutoring available in English, Hindi, Tamil, and more.
- **📊 Smart Analytics**: Track your progress with cognitive scoring and personalized mastery insights.
- **♿ Inclusive Design**: Accessibility-first UI tested for screen readers and voice-guided workflows.

---

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Pragathija/vorasenseai.git
   cd vorasenseai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

---

## 🛠️ Deployment

To deploy this project to **GitHub Pages**, simply run:

```bash
npm run deploy
```

Then, go to **Settings > Pages** in your GitHub repository and set the branch to `gh-pages`.
